# ARTIFICIALTECHNOLOGY

Modern AI-themed portfolio website built with Next.js and Tailwind CSS.

## Live Website

- Production: https://artificialtechnology.vercel.app

## Features

- Futuristic UI with animated sections and interactive effects
- AI scan-style About section with moving scan line
- Animated Experience Database section with floating chips and orbiting indicator
- Moving Tools Inventory rows with icon marquee animation
- Responsive layout for mobile, tablet, and desktop
- Vercel Analytics + Speed Insights integration

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS 3
- Vercel (Deployment, Analytics, Speed Insights)

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Open:

```text
http://localhost:3000
```

## Build & Run

Build for production:

```bash
npm run build
```

Run production server locally:

```bash
npm run start
```

## Project Scripts

- `npm run dev` - Run development server
- `npm run build` - Create production build
- `npm run start` - Start production server
- `npm run lint` - Run lint checks

## Project Structure

- `src/app` - App Router pages and global styles
- `src/components` - Reusable UI sections/components
- `public/images` - Logos, avatars, and tool icons
- `public/content` - Portfolio and project media assets

## Deployment

This project is deployed on Vercel and connected to this GitHub repository.
Pushing to `main` can be redeployed via Vercel CLI or Vercel dashboard.
